A comprehensive text styling plugin.

RGB Support

Bukkit color codes and RGB color codes are supported for all commands.

Nicknames

Pushes name customization further, allowing different names, and multiple different colors.

(Requires permission namecolor.nick)

Customizable

The prefix, and nearly all messages are customizable in the config file.

Styles

Name colors are customizable with bold, underline, italic, magic, and strikethrough text.

Moderation

The /whois command can reveal the user behind any nickname.

A nickname blacklist is planned for a future update.

Permissions

namecolor.* - all permissions

namecolor.set - allows players to set their own name color

namecolor.set.others - allows setting other player's name color

namecolor.nick - allows players to set their own nickname

namecolor.nick.others - allows setting other player's nicknames

namecolor.whois - allows use of the /whois command

bStats

Metrics are enabled by default. You can opt-out in your plugins folder at any time.
